# 🐛 Smart Maggot Web Group 2

## Sistem Monitoring Budidaya Maggot Berbasis IoT dan Web Dashboard

---

## 📌 Tentang Project

Smart Maggot Web Group 2 adalah sistem berbasis web yang digunakan untuk melakukan monitoring budidaya maggot secara **real-time** menggunakan teknologi Internet of Things (IoT).

Sistem ini mengintegrasikan **ESP32 + DHT22 sensor** untuk membaca suhu dan kelembaban, kemudian data ditampilkan ke dalam **web dashboard interaktif**.

Project ini mendukung konsep:
> 🌱 Smart Farming berbasis IoT, data, dan otomatisasi

---

## 🎯 Tujuan

- Membangun sistem monitoring budidaya berbasis IoT  
- Menampilkan data lingkungan secara real-time  
- Meningkatkan efisiensi budidaya maggot  
- Implementasi konsep smart farming  
- Media pembelajaran IoT dan web system  

---

## ⚙️ Fitur Utama

- 📊 Dashboard monitoring real-time  
- 🌡️ Monitoring suhu & kelembaban  
- 📈 Visualisasi data (chart & statistik)  
- 🤖 Sistem otomatis (kipas & lampu)  
- 📚 Halaman edukasi  
- 📑 Laporan data monitoring  
- 📱 Responsive design (mobile & desktop)  

---

## 🧠 Teknologi yang Digunakan

- HTML5  
- CSS3  
- JavaScript  
- ESP32 (IoT Microcontroller)  
- Sensor DHT22  
- Chart.js
